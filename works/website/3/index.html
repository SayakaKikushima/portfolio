株式会社ユニゾン・テクノロジー様
オウンドメディア「ユニゾンキャリア」 コンテンツ資料図解

Application
Scope
画像SEO対策、ラフ、VI制作、ビジュアル制作
Outline
株式会社ユニゾン・テクノロジー様が運営するオウンドメディア「ユニゾンキャリア」の記事内で掲載する使用制作を担当しました。
エンジニアになりたい求職者へ向け、業界、職種、働き方等情報を図解で提供し、ユーザーのメディアへの信頼を高めることが求められました。
Target
メインターゲット：20代男性/女性 業界未経験でエンジニア転職を目指す求職者
サブターゲット：20代男性/女性IT業界の経験者だが知識が浅く微経験である人
Design details
  • デザインのテンプレートを作成しブランドイメージの統一感を図りました。
  • イラストを交えながら未経験でもITの知識が身につくように意識しました。
  • Google画像検索から保存されることを想定し企業ロゴを入れ、企業ブランドの認知拡大を目指し、オウンドメディアとしての利益向上を計りました。
