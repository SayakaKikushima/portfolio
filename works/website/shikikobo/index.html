株式会社四季工房様
住宅展示場LP

Application
Scope
制作進行、顧客折衝、コンセプト設計 、ムードボード制作、ワイヤーフレーム制作、デザインカンプ制作
Outline
株式会社四季工房様が展示する住宅への見学予約数増加を目的としたLP制作を担当しました。
流入元:リスティング広告
目的:住宅展示場の予約数増加
Target
50~60代の子育てが終わり、子供が巣立った夫婦世代、第二の人生を検討している年齢層
Design details
  • ブランディング目的のLPと特殊だったため、CVを狙うのではなくいかに販売する住宅を魅了的かつ、四季工房の大切にしているメッセージを伝えていくのかを重視しデザインを制作
  • 大きく写真を使った構図、段々と写真とキャッチコピーが表示されるような時間軸をみせ、ゆったりとした暮らしをイメージしました。
  • 全体として極端なあしらいの使用を避け、落ち着いた配色、漆喰のテクスチャを用いることで、高級感が漂う雰囲気が伝わるようデザインをしました。
  • LPの結果、LPより住宅展示場への予約を獲得することに成功しました。
