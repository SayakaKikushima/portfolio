BrandDesign

株式会社BLAM様創業10周年イベントポスター・カルチャーブック（社内制作）

Application
Scope
印刷所選定、企画、コピー作成、写真撮影、デザイン制作
Outline
Design details
  • 創業10周年を記念したイベント開催のため、創業当時から現在までの沿革を記載するポスター、会社のカルチャーや事業を紹介した冊子の印刷所選定からディレクションおよび制作を担当しました。
  • ポスターはただ眺めるのみならず、過去10年間の写真を貼ったり、10年後の未来に挑戦したいことを書き込めるようにすることで「体験型」のクリエイティブとし、参加者のコミュニケーションツールになる狙いを込めました。
